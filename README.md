## Features
* Sign up / Sign in
    * Facebook Sign In
    * Google Sign In
    * Apple Sign In
    * Email / Pass Registration / Log In
* Real Time Conversations
* Message Type
    * Text Messages
    * Photo Messages
    * Video Messages
    * Location Messages
        * Location Navigation
    * Audio Messages
* Search for Users
* Deleting Conversations
* User Profile
    * User Name
    * Support
        * App Info
        * Privacy Policy
        * Email Contact
* Dark Mode Support
* FCM Push Notificaiton
